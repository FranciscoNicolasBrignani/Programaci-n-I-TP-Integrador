# Programaci-n-I-TP-Integrador

Tp integrador gestión de datos de países en Python:

Este proyecto es una aplicación de consola desarrollada en Python 3.x para gestionar información sobre países. Permite cargar datos desde un archivo CSV, agregar un pais, actualizar su poblacion y su superficie, realizar búsquedas, aplicar filtros, ordenar registros y generar estadísticas clave a partir del conjunto de datos. Utilizando listas, diccionarios y funciones.
Para poder gestionar datos de los países el usuario va a poder ingresar el nombre del país que quiera y obtendra los siguientes datos:
Nombre
Población
Superficie
Continente
Todo en lectura desde un archivo CSV que funciona como el almacenamiento de los datos
del proyecto.

Instrucciones de uso:
Para poder utilizar el programa es muy sencillo debe ejecutar el mismo y una vez hecho esto
aparece el siguente menu:
Menú de opciones:
1. Agregar un país
2. Actualizar los datos de población y superficie de un país
3. Buscar país
4. Filtrar países
5. Ordenar países
6. Mostrar estadísticas
7. Salir
Seleccione una opción (1-7):

Cada numero es una opcion para aceder a dicha opcion ingresamos por consola el numero de la opcion
que querramos y listo.
